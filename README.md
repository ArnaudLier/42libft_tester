# 42 Libft Tester

To use it, create a symbolic link to `libft.h` and `libft.a` in this directory.
Then, run:

```bash
make && ./main
```

If Libft has the bonus functions, run:
```bash
make bonus && ./main
```

Bonus functions' prototypes must be included in `libft.h`.
